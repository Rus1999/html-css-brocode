### Structure
```html
<!DOCTYPE html>
<html>
	<head>
		<title>Website Title</title>
	</head>
	<body>
		...
			content
		...
	</body>
</html>
```

### Header
```html
<h1> text </h1>
<h2> text </h2>
<h3> text </h3>
<h4> text </h4>
<h5> text </h5>
<h6> text </h6>
```

### Paragraph
```html
<!-- ignore space and new line -->
<p> text </p>

<!-- preformated include all space and new line -->
<pre> text </pre>
```

### Self Closing tag
```html
<!-- line break -->
<br>

<!-- horizontal rule -->
<hr>


```
